###MindLite

#Overview
Alzheimer’s disease causes gradual decline in memory and cognitive abilities MindLite enables early detection through continuous cognitive assessment Uses clinically inspired tests (MOCA-based) to evaluate patient performance 

#Cognitive Assessment Games
Family Recognition – Identify known faces and relationships 
Pattern Recognition – Predict sequences (tests logical memory) 
Memory Match – Match symbol pairs (speed + recall ability) 
Word Recall – Remember and reproduce words after delay
Reaction-Time – Click the button as soon as it turns green.

#Scoring & Analysis
Generates a Z-Score to quantify cognitive performance  Tracks performance over time (minimum 10 days) 
to detect: 
Sudden drops  
Gradual decline 

#Smart Alert System
Identifies abnormal cognitive decline patterns 
Alerts the Healthcare professionals who then analyse and choose whether to call the guardians.
